Cool HTML Tags

The marquee tag scrolls text across the screen.

The abbr helps the user to know what an acronym means (hover over abbr).


The del tag marks text out (delete) (strike tag does the same thing).


The ins tag inserts text by underlining it.


The datalist tag allows for a dropdown menu in a text box.


The mark tag highlights text


The optgroup tag creates a header for listed options
